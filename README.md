# 성경 암송 디스코드 봇

팀(7-8명)의 성경 암송 습관을 관리하는 Discord 봇 시스템.

## 핵심 기능
- 매일 DM 알림 (동기부여 말씀 + 오늘의 암송 계획)
- 버튼 1탭 응답 (최대 2탭, 타이핑 제로)
- Spaced Repetition 기반 자동 복습 구절 선택
- 주간 이미지 진도표 자동 생성
- 완료만 채널 공유, 미완료는 비공개 (부담 없는 구조)

## 커리큘럼
| 코스 | 구절 수 |
|------|---------|
| 입문 | 5 |
| 기초 | 8 |
| 성장 | 60 |
| 심화 | 242 |
| 마스터 | 180 |

- 주간: 새 구절 2개 배정
- 매일: 개인 설정 목표만큼 복습 (봇이 최적 구절 자동 선택)

## 기술 스택
- Node.js + discord.js v14
- better-sqlite3 (DB)
- node-canvas (이미지 생성)
- node-cron (스케줄링)
- 호스팅: Railway (무료)
