암송 시스템 이해하기

봇은 두 가지 트랙을 동시에 운영합니다. 새 구절을 배우는 트랙과, 이미 외운 구절을 반복하는 복습 트랙입니다.

🆕
새 구절 트랙
매주 월요일에 새로운 구절이 배정됩니다.

한 주 동안 매일 같은 구절이 알림으로 옵니다. 7번 반복하면 자연스럽게 익힐 수 있습니다.

다음 주 월요일에 새로운 구절이 배정됩니다.
→
🔄
복습 트랙
이미 배운 구절을 1번부터 순서대로 순환 복습합니다.

하루 목표에 맞게 매일 정해진 수의 구절이 알림으로 옵니다.

마지막 구절까지 완료하면 다시 처음부터 반복됩니다.
+
🔥
스트릭
새 구절 완료 + 복습 완료를 모두 마치면 스트릭이 1 증가합니다.

연속으로 쌓일수록 하나님 말씀이 마음 깊이 새겨집니다.

스트릭은 /진도에서 확인할 수 있습니다.

명령어 참고

슬래시 명령어 전체 정리

디스코드 채팅창에 / 를 입력하면 사용 가능한 명령어 목록이 자동으로 나타납니다.

/등록
암송 팀에 등록
처음 봇을 사용할 때 실행합니다. 코스 선택 → 시작 위치 → 하루 목표를 순서대로 설정하면 다음 날 아침부터 알림이 시작됩니다. 이미 등록한 분은 사용할 수 없습니다.
/설정
세부 설정 변경
등록 후 언제든지 코스, 시작 위치, 복습 목표, 알림 여부, 주일 알림 등을 변경할 수 있습니다. 변경 즉시 저장되며 다음 알림부터 적용됩니다.
/진도
암송 진도 확인
나의 완료 구절 수, 스트릭(연속 기록), 섹션별 진도를 한눈에 확인합니다. 팀 현황 보기 버튼으로 팀 전체 진도도 볼 수 있습니다.
/도움말
사용법 안내
봇의 기본 사용법과 명령어를 빠르게 확인할 때 사용합니다. 이 설명서가 없을 때 디스코드 안에서 바로 도움을 받을 수 있습니다.
